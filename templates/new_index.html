Visual Question Answering

What is VQA?

Visual Question Answering (VQA) is a computer vision task where a system is given a text-based question about an image, and it must infer the answer. Questions can be arbitrary and they encompass many sub-problems in computer vision, e.g.,

- Object recognition - What is in the image?
- Object detection - Are there any cats in the image?
- Attribute classification - What color is the cat?
- Scene classification - Is it sunny?
- Counting - How many cats are in the image?

Our Project

Our project builds a model based on VQA that generates a natural language answer, given an image and a natural language question about the image.
We have used VGG19 net for extracting the image features. A Baseline model and a different Co-Attention model is built to check the accuracy of the approach. The novel co-attention model for VQA jointly reasons about image and question attention.
The model reasons about the question (and consequently the image via the co-attention mechanism) in a hierarchical fashion via a novel 1-dimensional convolution neural network (CNN). It has a hierarchical architecture that co-attends to the image and question at three levels:
(a) word level
(b) phrase level
(c) question level.
We connect the image and question by calculating the similarity between image and question features at all pairs of image-locations and question-locations.
For the purpose of visualization, we have used the Python module Flask to integrate the web server.

Applications

Our model is helpful in assisting visually impaired individuals in understanding contents of images from the web.
It is also useful in Visual Turing Test or other computer vision tasks to evaluate the system’s ability when comparing with human performance.
It can be used in making decisions based on large quantities of surveillance data (“What kind of car did the man in the red shirt drive away in ?”)
Another application could be in interacting with robots (“Is my laptop in my bedroom upstairs ?”)
